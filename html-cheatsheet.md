
# 🌐 HTML Cheat Sheet

## 📄 Basic Structure
```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Page Title</title>
</head>
<body>
  Content goes here
</body>
</html>
````

---

## 🏷️ Common Tags

### Headings

```html
<h1>H1</h1>
<h2>H2</h2>
<h3>H3</h3>
<h4>H4</h4>
<h5>H5</h5>
<h6>H6</h6>
```

### Paragraph & Text

```html
<p>Paragraph</p>
<br> <!-- Line break -->
<hr> <!-- Horizontal line -->
<strong>Bold</strong>
<em>Italic</em>
```

---

## 🔗 Links & Media

### Links

```html
<a href="https://example.com">Click here</a>
```

### Images

```html
<img src="image.jpg" alt="description" width="300">
```

### Video

```html
<video controls>
  <source src="movie.mp4" type="video/mp4">
</video>
```

---

## 📋 Lists

### Unordered List

```html
<ul>
  <li>Item</li>
  <li>Item</li>
</ul>
```

### Ordered List

```html
<ol>
  <li>First</li>
  <li>Second</li>
</ol>
```

---

## 📊 Tables

```html
<table>
  <tr>
    <th>Header</th>
    <th>Header</th>
  </tr>
  <tr>
    <td>Data</td>
    <td>Data</td>
  </tr>
</table>
```

---

## 🧾 Forms

```html
<form>
  <input type="text" placeholder="Name">
  <input type="email" placeholder="Email">
  <input type="password" placeholder="Password">
  <button type="submit">Submit</button>
</form>
```
