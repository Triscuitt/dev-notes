## Sorting Algorithms
- **Bubble Sort** – Simple comparison-based sort; slow for large lists.
- **Insertion Sort** – Builds sorted list one element at a time; good for small arrays.
- **Merge Sort** – Divide-and-conquer sort; efficient O(n log n) performance.
- **Quick Sort** – Fast divide-and-conquer sort; good average-case speed.
- **Selection Sort** – Selects smallest element repeatedly; simple but slow.

## Searching Algorithms
- **Linear Search** – Checks each element one by one; simple but slow for large lists.
- **Binary Search** – Efficient search on sorted data; halves search space each step.
- **Jump Search** – Searches sorted lists in fixed steps; faster than linear.

## Recursion / Backtracking
- **Factorial** – Multiplies numbers from 1 to n recursively.
- **Fibonacci** – Generates nth Fibonacci number recursively.
- **Tower of Hanoi** – Solves puzzle recursively by moving disks between pegs.
- **N-Queens** – Places N queens on a board without conflicts (backtracking).
- **Permutations** – Generates all possible arrangements recursively.
